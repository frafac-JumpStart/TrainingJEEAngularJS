• Protractor is the E2E framework test for Angular
  • Built on WebdriverJS aka Selenium Server
  • Allows running tests targeting remote addresses
  • Has it's own runner, no need for Karma

Getting started

> # 1) INSTALL PROTRACTOR IN LOCAL MODULES > npm install protractor #local install > # 2) JUST DOWNLOAD SELENIUM SERVER LOCALLY > node_modules/protractor/bin/webdriver-manager update > # 3) FOR OUR PRACTICAL WORKS ONLY, MAKE THE STANDALONE AVAILABLE IN THE config/scripts FOLDER > mkdir config/scripts && cp -r node_modules/protractor/selenium/* config/scripts/ /scripts/
  • For PW, grunt test:e2e should be enough, be sure that your node server is started
