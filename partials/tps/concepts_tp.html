1. Get following AngularJS application war.
  2. Extract the archive on your filesystem
  3. In your project directory, open a new console and execute these commands:
    • mvn jetty:run: Build & Run a server to access the application
  4. Product description page is handled by two files: the template detail.html and the controller DetailController.js. Use the Angular two way binding to fill the template with the product informations.
  5. Test your application at following url: http://localhost:8080/bookcat#/book/1
