1. Get and install Node.js.
  2. Get following AngularJS application skeleton.
  3. Extract the archive on your filesystem
  4. In your project directory, open a new console and execute these commands:
    • npm install: Install server side dependencies
    • npm install -g bower: Install bower globally
    • bower install: Install client side dependencies
    • node server: Run a server to access the application
  5. remark: If you have to set the proxy, then define HTTP_PROXY and HTTPS_PROXY environment variables
  6. Product description page is handled by two files: the template detail.html and the controller DetailController.js. Use the Angular two way binding to fill the template with the product informations.
  7. Test your application at following url: http://localhost:3000/#/book/1
