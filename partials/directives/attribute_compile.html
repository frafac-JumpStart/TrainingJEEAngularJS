• Used to transform the template DOM before it is compiled.
  • Can have optional following parameters (identified by their order):
    1. element: Template element.
    2. attrs: Template attributes.
    3. transclude: Transclude function.
  • The link attribute is ignored as soon as the compile function is defined, but compile can return link if needed
